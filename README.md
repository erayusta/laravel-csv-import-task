## Setup Process
 
1.
```
git clone git@github.com:erayusta/laravel-csv-import-task.git
```
2.
```
composer install
```
3.
```
php artisan storage:link
```

4.
```
change database information .env file
```

5.
```
php artisan key:generate
```


6.
```
php artisan migrate
```
